# PH6-L3D-003 — Correção de estratégia de camadas

## Escopo
- Revisão da hierarquia de camadas para conflitos entre header fixo, menu mobile e overlays de modal/lightbox.
- Garantia de bloqueio de interação de fundo no menu mobile.
- Ajuste de prioridade visual do lightbox acima de elementos fixos.

## Alterações implementadas
- Introduzidos tokens globais de layering (`--layer-*`) em `globals.css` para padronizar z-index por tipo de camada.
- `HeaderNavigation` atualizado para:
  - usar token de camada do header;
  - aplicar backdrop e painel mobile em `position: fixed` com cobertura total da viewport abaixo do header;
  - bloquear scroll de fundo via `document.body.style.overflow = "hidden"` enquanto menu estiver aberto;
  - fechar por `Escape`;
  - expor `aria-expanded` e `aria-controls` no botão de toggle.
- `Gallery` atualizado para usar token de modal (`--layer-modal`) no lightbox, garantindo prioridade acima de header/menu fixos.
- `layout.tsx` ganhou wrapper `<main id="app-content">` para delimitar conteúdo principal.
- Teste de regressão criado para o fluxo do menu mobile (`HeaderNavigationLayering.test.tsx`).

## Resultado
- Menu mobile passa a cobrir corretamente a interface de conteúdo e bloquear interação/scroll de fundo enquanto aberto.
- Lightbox permanece em camada superior ao header fixo.
